# AI News Scraper

特定のニュースサイトからAI関連の最新情報を収集し、自動的に記録するプロジェクトです。

## 概要
GitHub Actionsを用いて、PCをオフラインにしていても毎日自動でニュースをスクレイピングし、CSVファイルに記録・蓄積します。AI分野の最新動向を効率的に追いかけるためのデータ収集基盤です。

## 主な機能
- **自動データ収集**: Pythonスクリプトを用いて、ターゲットサイトからニュースタイトルとURLを抽出。
- **定期実行**: GitHub Actionsにより毎日定時に自動実行。
- **データ蓄積**: 収集したデータを `news_log.csv` に自動追記・保存。

## 使用技術
- Python
- BeautifulSoup (Webスクレイピング)
- GitHub Actions (自動実行環境)
